Security by architecture

Your source code is not telemetry.

CodeSpace is local-first and provider-independent. It does not require embeddings, a model account, or a hosted database.

✓

Local processingNo source upload path in the binary.

✓

Secret redactionCredential-like values are removed before context and MCP reads.

✓

Path confinementSymlinks and metadata escape paths are blocked.

✓

Loopback RESTRemote exposure is explicit and warned.
