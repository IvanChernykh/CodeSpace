# Third-party provenance policy

CodeSpace 1.0.0 is an independent clean-room implementation. The release contains no copied source from the projects listed in the original product brief and has no runtime crate dependencies.

The referenced projects were treated as product research: feature categories, interface ideas, and publicly described workflows. Before any future code import, maintainers must record:

1. upstream repository and immutable commit;
2. exact file-level license and copyright notices;
3. compatibility decision with Apache-2.0;
4. required attribution/NOTICE text;
5. security review and maintained fork strategy;
6. generated SBOM entry.

Apache-2.0 on this repository does **not** grant permission to copy code from a differently licensed upstream project. Each upstream license controls that upstream code.
